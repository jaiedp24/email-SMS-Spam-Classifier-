# email-SMS-Spam-Classifier-
##  Spam Classifier App

A Machine Learning web application that classifies SMS messages as **Spam** or **Not Spam** using Natural Language Processing (NLP) and the Multinomial Naive Bayes algorithm.

##  Live Demo

https://vcc85nfckjccajr8m5tdcv.streamlit.app/

## 🛠️ Technologies Used

* Python
* Scikit-learn
* Pandas
* NumPy
* NLTK
* Streamlit

## 🧠 Model

* Multinomial Naive Bayes
* TF-IDF Vectorizer

## 📂 Project Files

* `app.py` — Streamlit web application
* `model.pkl` — Trained machine learning model
* `vectorizer.pkl` — TF-IDF vectorizer
* `requirements.txt` — Project dependencies

## ▶️ Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
